## LACOSTE

- Одежда, обувь, парфюмерия, часы, очки
- https://lacoste.ru/api/catalog.php?
- Параметры поиска:

1. q = < Ключевое слово >
2. perPage = < Количество товаров - результатов поиска >
3. sort = < default / name / price >
4. order = < desc / asc >
